# hello_world: Efficiency at its Greatest

Welcome to hello_world! This project is a fun demonstration of the power of Rust. I've deliberately used as many as possible to showcase the versatility of the language and the efficiency of the Rust ecosystem.

## Installation

Before getting started, make sure you have Rust installed on your machine. You can install Rust by following the instructions on the [official Rust website](https://www.rust-lang.org/tools/install).

Once Rust is installed, simply clone this repository and navigate to the project directory in your terminal.

```bash
git clone https://github.com/guymarshall/hello_world.git
cd hello_world
cargo run #testing
cargo run --release #blazing-fast performance
